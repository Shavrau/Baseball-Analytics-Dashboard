# Baseball Analytics Dashboard ⚾

An interactive **Power BI** dashboard created to explore historical baseball statistics and identify trends in player performance, team results, and offensive production across different seasons.

The project uses the **Lahman Baseball Database**, covering data from **1991 to 2025**, and provides an intuitive analytical experience through KPIs, rankings, and interactive visualizations.

---

## 🎯 Project Objective

The goal of this project is to transform historical baseball data into actionable insights through interactive dashboards, allowing users to:

* Analyze player performance over time
* Explore Home Run and RBI trends
* Identify historical leaders and standout seasons
* Compare performance across different eras
* Monitor key baseball statistics through visual analytics

---

## 📊 Dashboard Highlights

* Global KPI overview
* Historical Home Run trends
* Career leader rankings
* Player performance analysis
* Interactive year filtering
* Dynamic data exploration

## 📖 Data Dictionary

### Batting

| Column | Description    |
| ------ | -------------- |
| AB     | At Bats        |
| R      | Runs Scored    |
| H      | Hits           |
| HR     | Home Runs      |
| RBI    | Runs Batted In |
| SB     | Stolen Bases   |
| SO     | Strikeouts     |

### People

| Column       | Description              |
| ------------ | ------------------------ |

| bats         | Batting hand             |

### Teams

| Column     | Description       |
| ---------- | ----------------- |

| lgID       | League identifier |

### Parks

| Column    | Description        |
| --------- | ------------------ |
